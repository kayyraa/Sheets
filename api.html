    Usage

    Commands

  • Simple, self-explanatory functions that perform an action without returning a value.
    • Example:
    import
    *
    as
    Api
    from
    "./api.js"
    ;
    Api
    .
    IsMobile
    ()
    ;
    // Checks if the user device is mobile and returns true or false.

    Functions

  • Functions that accept parameters and return values like booleans, strings, numbers, or HTML content.
    • Example:
    import
    *
    as
    Api
    from
    "./api.js"
    ;
    Api
    .
    Sheet
    .
    GetItem
    (
    "D"
    ,
    12
    ,
    9
    )
    ;
    // Fetches a specific item from a grid using coordinates.

    Classes

  • Collections of related commands, functions, and properties grouped together.
    • Example:
    import
    *
    as
    Api
    from
    "./api.js"
    ;
    Api
    .
    Sheet
    .~
    // Classes can contain commands, functions, variables, and nested classes.

    Syntax

  • The API syntax is intuitive, organized from the most complex to the simplest concepts.
    • Example:
    import
    *
    as
    Api
    from
    "./api.js"
    ;
    // Syntax follows: Script - Class - Children - Descendant
    Api
    .
    Sheet
    .
    Grid
    .
    Remove
    ()
    ;

    DOM

  • Access and manipulate the Document Object Model (DOM) using the API.
    • Example:
    import
    *
    as
    Api
    from
    "./api.js"
    ;
    Api
    .
    Grid
    .
    cloneNode
    ()
    ;
    // Accesses and clones the "Grid" element defined in api.js.
